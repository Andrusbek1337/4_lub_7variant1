# Описание лабораторной работы:

1. Класс Time
#### Поля:
    hours - часы (0-23)
    minutes - минуты (0-59)

Основные методы:
#### Конструкторы:
    Time() - конструктор по умолчанию (00:00)
    Time(short int h, short int m) - конструктор с параметрами
    Time(const Time& other) - конструктор копирования

#### Методы доступа (геттеры/сеттеры):
    getHours(), getMinutes() - получение значений
    setHours(), setMinutes() - установка значений с проверкой

#### Основные методы:
    subtract() - вычитание одного времени из другого
    normalize() - нормализация времени (приватный метод)
    isValidTime() - проверка корректности времени

Перегруженные операторы:
#### Унарные операции:
    operator++() и operator++(int) - добавление минуты
    operator--() и operator--(int) - вычитание минуты

#### Операции приведения типа:
    operator int() - неявное приведение к количеству минут
    operator bool() - явное приведение к bool (true если не 00:00)

#### Бинарные операции:
    operator<() - сравнение "меньше"
    operator>() - сравнение "больше"
    operator==() - проверка равенства
    operator!=() - проверка неравенства

#### Оператор вывода:
    operator<< - форматированный вывод в формате HH:M

Вспомогательные функции:
#### inputNumber()
    Функция для безопасного ввода чисел с проверкой:
    Проверяет, что ввод состоит только из цифр
    Проверяет диапазон значений
    Защищает от ввода букв и некорректных символов
    Повторяет запрос при ошибке ввода

#### inputTime()
    Функция для ввода времени с клавиатуры:
    Использует inputNumber() для ввода часов и минут
    Создает объект Time с введенными значениями




Как это работает:

Инициализация

#### Тестирование конструкторов
    Создание объекта с конструктором по умолчанию (00:00)
    Создание объекта с вводом параметров от пользователя
    Создание копии объекта

#### Тестирование геттеров и сеттеров
    Пользователь вводит значения для часов и минут
    Демонстрируется работа сеттеров и геттеров

#### Тестирование оператора вывода
    Создание объекта с фиксированным временем (05:07)
    Демонстрация форматированного вывода

#### Тестирование метода вычитания
    Пользователь вводит два времени
    Выполняется вычитание с учетом перехода через полночь
    Выводится результат

#### Тестирование унарных операций
    Пользователь вводит время
    Демонстрируется работа операторов ++ и --
    Показывается разница между префиксной и постфиксной формами
